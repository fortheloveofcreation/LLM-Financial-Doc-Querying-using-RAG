LLM based Chatbot for financial document query

1.Created a RAG based chatbot using LangChain to analyze financial documents. Utilized the Llama 3.1 LLM and HuggingFace's all-MiniLM-L6-v2 embedding model.

2. Implemented document loading for PDFs and text with PyPDFLoader, followed by text segmentation, embedding creation, and indexing within a vector database to optimize information retrieval.

Technologies: Python, LangChain, Llama 3.1, Chroma
